### What I'm working on
- **WFTAF** — a modular DFIR framework (PowerShell + Python) for fast triage of
  Windows endpoints, built around real ATT&CK persistence and lateral-movement
  techniques.
- **A detection lab in Azure** — simulating adversary techniques with Atomic Red
  Team, piping telemetry into Sentinel, and writing detections that actually
  fire on them.
- Generally: turning messy Windows logs into telemetry you can trust, and
  detections that hold up outside a slide deck.
